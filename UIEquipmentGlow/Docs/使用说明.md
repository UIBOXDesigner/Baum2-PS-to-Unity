# UI 高级装备框光效资源包使用说明

## 一、资源内容

导入后目录为：

```text
Assets/UIEquipmentGlow
├─ Shaders
│  ├─ UI_Equipment_OuterGlow.shader        // 外圈发光，支持呼吸
│  ├─ UI_Equipment_FrameShine.shader       // 边框扫光/流光
│  └─ UI_Equipment_InnerRarity.shader      // 物品底图品质色晕染
├─ Scripts
│  ├─ UIEquipmentFrameEffect.cs            // 运行时品质光效控制组件
│  ├─ UIEquipmentFrameQualityBinder.cs     // 按钮/事件绑定用
│  ├─ UIEquipmentFrameRuntimeExample.cs    // 运行时调用示例
│  └─ UIEquipmentSparkFX.cs                // 轻量 UI 星点特效，不依赖第三方粒子插件
├─ Editor
│  └─ UIEquipmentGlowEditorTools.cs        // 一键生成材质球与示例节点
├─ Textures
│  ├─ item_bg.png                          // 示例底图
│  ├─ frame_mask.png                       // 示例边框遮罩
│  ├─ glow_frame.png                       // 示例发光边框
│  └─ spark_dot.png                        // 示例 UI 星点
└─ Materials                               // 通过菜单自动生成默认材质球
```

## 二、导入方式

把 `Assets/UIEquipmentGlow` 文件夹复制到 Unity 项目的 `Assets` 目录下。

等待 Unity 编译完成后，顶部菜单会出现：

```text
Tools / UI Equipment Glow
```

先点击：

```text
Tools / UI Equipment Glow / 1. Generate Default Materials
```

再点击：

```text
Tools / UI Equipment Glow / 2. Create Demo Item Cell
```

场景中会生成一个 `Demo_Legendary_EquipmentCell` 示例节点。

## 三、推荐节点结构

正式项目建议按下面结构做物品格：

```text
ItemCell
├─ Bg               // 物品底图，可挂 Inner Rarity Tint 材质
├─ Glow             // 外发光层，尺寸比 Frame 大 10%~25%
├─ Frame            // 品质边框
├─ Icon             // 装备图标
├─ Shine            // 流光层，使用 Frame 同样的遮罩图
└─ ParticleFX       // 可选，只给神话/红装/获得展示开启
```

关键点：`Glow` 必须比 `Frame` 稍大，否则外发光会被 UI 图片矩形区域裁掉，看起来像没有向外扩散。

## 四、组件挂载方式

在 `ItemCell` 根节点上挂：

```text
UIEquipmentFrameEffect
```

然后拖引用：

```text
Background Image  -> Bg
Frame Image       -> Frame
Glow Image        -> Glow
Shine Image       -> Shine
Pulse Target      -> Glow 的 RectTransform，不要绑定 ItemCell 根节点
Particle FX       -> 可选 ParticleSystem
Particle FX Root  -> ParticleFX_GoldSpark 或你的 UI 粒子根节点
```

材质引用：

```text
Outer Glow Material   -> M_UI_Glow_Legendary 或其他 Glow 材质
Frame Shine Material  -> M_UI_FrameShine_Gold
Inner Rarity Material -> M_UI_InnerRarity_Default
```

运行时切品质：

```csharp
using UIEquipmentGlow;

public class Example
{
    public UIEquipmentFrameEffect effect;

    public void Refresh(int quality)
    {
        effect.ApplyQuality((UIEquipmentQuality)quality);
    }
}
```

品质枚举：

```text
0 Normal
1 Rare
2 Epic
3 Legendary
4 Mythic
```

## 五、品质表现规则

默认表现如下：

```text
Normal：普通边框，不开光效
Rare：弱外发光
Epic：外发光 + 呼吸缩放
Legendary：外发光 + 呼吸缩放 + 边框扫光
Mythic：强外发光 + 呼吸缩放 + 边框扫光 + UI 星点特效开关
```

可以在 `UIEquipmentFrameEffect` 的 `Presets` 数组里调整每个品质的颜色、发光强度、发光范围、扫光速度、呼吸幅度。

本版已修正 Demo 中呼吸目标绑定错误：`Demo_Legendary_EquipmentCell` 根节点不再缩放，`pulseTarget` 默认绑定到 `Glow` 子节点。若你在旧场景里已经生成过旧 Demo，请删除旧的 `Demo_Legendary_EquipmentCell` 后重新执行 `Tools / UI Equipment Glow / 2. Create Demo Item Cell`。

`ParticleFX_GoldSpark` 是内置的 UI 星点特效节点，默认隐藏，切换到 `Mythic` 品质时会由 `Particle FX Root` 自动开启。它不依赖 `ParticleSystem` 或第三方 UI 粒子插件，因此不会再出现粒子资源缺失的问题。

## 六、Shader 参数说明

### UI/Equipment/Outer Glow

```text
_GlowColor      发光颜色
_GlowIntensity  发光强度
_GlowSize       发光范围
_PulseSpeed     呼吸速度
_PulseAmount    呼吸强度
_EdgeBoost      边缘增强
```

### UI/Equipment/Frame Shine

```text
_ShineColor      扫光颜色
_ShineIntensity  扫光强度
_ShineWidth      扫光宽度
_ShineSoftness   扫光边缘柔和度
_ShineSpeed      扫光速度
_ShineAngle      扫光角度
```

### UI/Equipment/Inner Rarity Tint

```text
_RarityColor       品质晕染色
_RarityStrength    晕染强度
_VignetteStrength  边缘暗角/色晕强度
```

## 七、性能建议

背包列表里不要让所有格子都开复杂效果。推荐规则：

```text
普通滚动列表：只开静态边框或弱 Glow
选中装备：开启 Glow 呼吸和 Shine
新获得高级装备：播放一次强表现
神话/红装：允许常驻轻量扫光，粒子尽量限制数量
不可见格子：关闭动画、粒子和高频刷新
```

`UIEquipmentFrameEffect` 默认会给材质创建运行时实例，避免多个物品格共用同一个材质时互相串色。
